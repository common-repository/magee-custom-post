== Description ==

### Overview 

Magee Custom Post is WordPress plugin that provides custom post types.


== Installation ==

Unzip plugin files and upload them under your '/wp-content/plugins/' directory.

Resulted names will be: './wp-content/plugins/mcp-shortcodes/*'

Activate plugin at "Plugins" administration page.
